# BookStore

A simple Book Store API with otp verification login

# Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/rahulkbharti/book-store-react.git
   cd book-store-react
   ```

2. **Install The Packeges:**

   ```bash
   npm install
   ```

3. **Start the application:**

   ```bash
   npm run dev
   ```

4. **Open in browser:**
   ```
   http://localhost:5173/
   ```
